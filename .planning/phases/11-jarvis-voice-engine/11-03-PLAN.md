# 11-03: Speech-to-Text Input & Mic Button

## Goal
Voice input via Web Speech API with mic button in ChatInput, recording state indicators, and wake word detection.

## Files to Create
- `jarvis-ui/src/hooks/useSpeechRecognition.ts` — Web Speech API wrapper with wake word detection

## Files to Modify
- `jarvis-ui/src/components/center/ChatInput.tsx` — Add mic button, voice input integration

## Design
- Web Speech API (SpeechRecognition) for zero-cost browser-native STT
- Continuous recognition mode with interim results
- Mic button toggles recording on/off
- Pulsing red indicator during recording
- "JARVIS" wake word triggers voice input automatically when enabled
- Transcript populates the text input, user confirms with Enter or auto-sends

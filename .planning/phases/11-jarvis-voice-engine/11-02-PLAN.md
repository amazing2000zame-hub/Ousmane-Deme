# 11-02: Frontend Audio Playback + Chat Integration

## Goal
TTS playback hook, play/stop buttons on assistant messages, auto-play on response completion.

## Files to Create
- `jarvis-ui/src/hooks/useVoice.ts` — TTS playback (fetch /api/tts, Web Audio API, browser fallback)

## Files to Modify
- `jarvis-ui/src/components/center/ChatMessage.tsx` — Add play/stop button on assistant messages
- `jarvis-ui/src/components/center/ChatPanel.tsx` — Auto-play integration on response complete
- `jarvis-ui/src/hooks/useChatSocket.ts` — Emit voice auto-play trigger on chat:done

## Design
- useVoice hook handles: fetch audio from backend, play via AudioContext, browser fallback
- Play button appears on all assistant messages with text content
- Auto-play triggers when voice is enabled + autoPlay setting is on
- AudioContext + AnalyserNode exposed for visualizer (11-03)
